# WebSocket 实时聊天功能使用说明

## 📦 安装依赖

在运行应用之前，需要安装 Flask-SocketIO 相关依赖：

```bash
pip install Flask-SocketIO python-socketio eventlet
```

或者使用 requirements.txt：

```bash
pip install -r requirements.txt
```

## 🚀 启动应用

```bash
python app.py
```

应用将在 `http://127.0.0.1:5000` 启动

## 💬 使用聊天功能

### 1. 开始聊天
- 浏览商品详情页
- 点击紫色的 "💬" 在线聊天按钮
- 自动跳转到与卖家的聊天页面

### 2. 查看聊天列表
- 点击右下角的紫色聊天图标（购物车旁边）
- 查看所有聊天记录
- 未读消息会显示红色徽章

### 3. 发送消息
- 在聊天页面输入框中输入消息
- 点击"发送"按钮或按 Enter 键发送
- 消息实时显示在聊天窗口

### 4. 消息状态
- ✓ 未读：消息已发送但对方未查看
- ✓ 已读：对方已查看消息（绿色）

## 🔧 技术实现

### 后端技术
- **Flask-SocketIO**: WebSocket 服务器
- **SQLAlchemy**: 消息数据存储
- **Room 机制**: 用户隔离的消息推送

### 前端技术
- **Socket.IO Client**: WebSocket 客户端
- **原生 JavaScript**: 无框架依赖
- **CSS3 动画**: 流畅的视觉效果

### 数据流程
1. 用户 A 发送消息 → Socket.IO 服务器
2. 服务器保存消息到数据库
3. 服务器推送消息给用户 B
4. 用户 B 接收并显示消息
5. 自动标记为已读并通知用户 A

## 📝 注意事项

1. **必须登录**: 只有登录用户才能使用聊天功能
2. **管理员限制**: 管理员账号不能使用聊天功能
3. **浏览器支持**: 需要支持 WebSocket 的现代浏览器
4. **网络要求**: 需要稳定的网络连接

## 🐛 常见问题

### Q: 消息发送失败？
A: 检查网络连接，确保 WebSocket 连接状态为"已连接"

### Q: 收不到实时消息？
A: 刷新页面重新建立 WebSocket 连接

### Q: 聊天图标不显示？
A: 确认已登录普通用户账号（非管理员）

### Q: 如何清除聊天记录？
A: 目前不支持删除聊天记录，消息会永久保存

## 🎨 自定义样式

聊天界面使用了渐变色主题，可以在以下文件中修改：

- `templates/chat.html`: 聊天页面样式
- `templates/chat_list.html`: 聊天列表样式
- `static/css/style.css`: 全局聊天图标样式

主要颜色：
- 主色调: `#667eea` (紫色)
- 渐变色: `#667eea` → `#764ba2`
- 未读徽章: `#e74c3c` (红色)
- 已读状态: `#4CAF50` (绿色)
